## 📖 프로젝트 개요

- **Opus**는 카카오의 이미지 생성 AI인 Karlo 모델을 활용한 이미지 공유 서비스입니다. <br><br>
- 사용자는 Karlo 모델을 사용하여 프롬프트를 포함한 다양한 설정을 통해 이미지를 생성하고 <br> 이를 다른 사용자들과 공유할 수 있습니다. <br><br>
- 사용자는 다른 사용자가 생성한 이미지를 보고 좋아요를 누르거나 댓글을 작성할 수 있습니다.
  <br> <br>
